

## 博芯获取统编名单列表


**接口地址**:`/v1/boXin/boXinRosterList`


**请求方式**:`GET`


**请求数据类型**:`application/x-www-form-urlencoded`


**响应数据类型**:`*/*`


**接口描述**: 基于学校ID和统编ID获取名单列表。响应中的businessId用于后续调用editionStudentZip下载素材，也可通过小程序分享直接传入。


**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|schoolId|schoolId|query|true|string||
|editionId|editionId|query|true|integer(int32)||
|editionType|editionType|query|true|integer(int32)||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|RestResult«List«BoXinStudentDTO»»|
|401|Unauthorized||
|403|Forbidden||
|404|Not Found||
